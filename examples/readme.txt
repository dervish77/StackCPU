StackCPU examples
=================

Here's a few example programs using the StackCPU assembly language.  These can the "assembled" 
using the tools found in the "src" folder.

Example 1 - prints the text "HELLO" to the serial output PR register

Example 2 - adds the numbers 1 thru 5 and outputs the sum to the OR register

Example 3 - adds the numbers 1 thru 10 in a loop and then outputs sum to the OR register
